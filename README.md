# it-start-platform-ayaz

Стек: React, TypeScript, Redux Toolkit, Chart.js / recharts, 
Tailwind CSS

Моковые данные придумай сам в src/lib/mock-data.ts

## Фича 1: Конструктор отчётов /reports

- Список готовых шаблонов отчётов:
  - Готовые к аутстаффу сотрудники
  - Прогресс команды за период
  - Активность менторов
  - Задачи по статусам
- Клик на шаблон → генерация отчёта
- Отчёт содержит: таблицу с данными + график
- Фильтры: период (неделя/месяц/квартал), 
  ментор, статус
- Кнопка "Экспорт CSV" — скачивает данные таблицы
- Кнопка "Печать" — window.print() с @media print

## Фича 2: Трекер аутстаффа /outstaff

- Список компаний-партнёров
  с карточками: название, контакт, статус партнёрства
- У каждой компании — открытые вакансии:
  грейд, стек, ставка, дедлайн
- Таблица предложений: какой разраб → 
  в какую компанию → на какую вакансию предложен
- Статусы предложения: 
  отправлен / на собесе / оффер / отказ
- Клик на строку → детали: 
  когда отправили, комментарии, следующий шаг
- Фильтрация по статусу и компании
## Требования

- Компоненты декомпозированы
- Redux Toolkit для глобального стейта
- TypeScript, без any
- Адаптивная вёрстка
- Каждая фича — отдельная ветка и PR в main
